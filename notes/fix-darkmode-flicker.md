# 🛠 브랜치명: fix/darkmode-flicker

## 📋 작업 요약
- 다크모드 적용 시 SSR(서버 사이드 렌더링)에서 테마 쿠키 미적용으로 발생한 FOUC(깜빡임) 문제 해결

## 🎯 수정 이유
- 페이지 초기 로딩 시 사용자 테마 설정(dark/light)이 제대로 적용되지 않아 깜빡이는 문제가 있었음

## ✅ 적용 방식
- 쿠키를 서버에서 읽어 초기 테마 설정
- SSR 단계에서 사용자 테마를 적용해 FOUC 방지
